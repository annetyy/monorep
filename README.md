# Setup

yarn install

# Run

## Primeiro rodar o watch

cd packages/common

yarn watch

## Depois rodar o APP ou WEB

### WEB

cd packages/web

yarn start

### APP

#### term 1

cd packages/common
yarn watch

#### term 2

cd packages/app
yarn start

#### term 3

cd packages/app
wml start

#### term 4

cd packages/app
react-native run-android
